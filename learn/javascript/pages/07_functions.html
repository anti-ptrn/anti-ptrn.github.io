توابع [1]

توابع بخشی جدایی ناپذیر از JS هستند. در ابتدا اجازه دهید با مهفوم تابع آشنا شویم. همانطور که قبلاً گفتیم هدف اصلی JS اضافه کردن قابلیت‌های پویا به صفحات وب است. قصد داریم امکاناتی به JS اضافه کنیم که در چندین صفحه مختلف یا چندین بار در یک صفحه استفاده شود؛ آیا باید آن تکه کد برای هر بار استفاده تکرار کنیم؟ به جای اینکه هربار قابلیت مورد نظر را پیاده‌سازی کنیم، می‌توانیم از مفهوم توابع استفاده کنیم و یکبار تابع مورد نظر را تعریف کنیم و هربار که به آن نیاز داشتیم آن‌را فراخوانی کنیم.

توابع داخلی زیادی در JS وجود دارند. علاوه بر این توابع، می‌توانیم خود را براساس نیاز پیاده‌سازی کنیم. جهت کارکرد توابع به تکه کد زیر دقت نمایید:

<!DOCTYPE html>
<html>
<head>
    <title> Java Script in a nutshell </title>
</head>

<body>
    <h1> Functions in JavaScript </h1>
    <p> Click on "Click Here" to execute the code in the function </p>

    <script type="text/javascript">
        function PacktPub() {
            var x = prompt("Hey", "");
            alert("Welcome to Packt, " + x);
        }
    </script>

    <a href="javascript:PacktPub()">Click Here</a>
</body>

</html>

در فیلم زیر، اجرای تکه کد بالا نمایش داده شده است:

در تکه کد بالا، تابعی به نام PacktPub تعریف شده است، که داخل آن از یکی از توابع داخلی JS یعنی تابع prompt جهت دریافت مقدار ورودی از کاربر استفاده شده است. سپس با استفاده از تابع داخلی alert یک پیام خوش آمد گویی نمایش می‌دهد. در ادامه کد اگر به کد <a href="javascript:PacktPub()">Click Here</a> دقت کنید، خواهید دید که متد PacktPub در آن فراخوانی شده است.

⚠️همانطور که قبلاً گفته شد، JS زبانی حساب به حورف کوچک و بزرگ[2] است. برای مثال اگر تابعی به نام SayHello تعریف کردید، نمی‌توانید موقع فراخوانی از عبارت sayhello استفاده کنید.

نحوه تعریف تابع

ساختار کلی تابع به شکل زیر است:

function نام_تابع(arg1, arg2, ... , arg_n){
    // دستورات تابع
}

هر تابع می‌توانید هیچ یا n ورودی[3] داشته باشد، برای درک بهتر موضوع به تکه کد زیر دقت کنید:

function sum(a, b){
    alert(a + b);
}

sum(1, 2); // alert(3)

در تابع بالا دو وردوی به نام a و b تعریف شده است و در زمان فراخوانی آن مقادیر متناظر با آنها به تابع ارسال شده اند.

مقدار بازگشتی تابع

همانطور که در مثال اول مشاهده کردید، تابع prompt مقداری را از کاربر دریافت می‌کند و آن مقدار به ما باز می‌گرداند. اما چطور می‌توان اینکار را در توابع خود پیاده سازی‌کنیم؟ در JS کلمه کلیدی به نام return وجود دارد که هر وقت تابع به آن برسد، علاوه بر اینکه از اجرای ادامه‌ی کد در JS جلوگیری می‌کند و از تابع خارج می‌شود، می‌توان با آن مقدار خروجی تابع را تعیین کرد.

در مثال قبلی؛ تکه کدی نوشتیم که مقدار a و b را از ورودی دریافت می‌کند و در پیامی مجموع آن‌ها را نمایش می‌دهد. قصد نوشتن تابعی داریم که مجموع a و b را بازگرداند:

function sum(a, b){
    var result = a + b;
    return result;
}

یا به شکل ساده تر:

function sum(a, b){
    return a + b;
}

بالاتر گفته شد که از کلمه کلیدی return برای خروج از تابع نیز می‌توان استفاده کرد، اما چطور؟ فرض کنید قصد نوشتن برنامه‌ای داریم که دو حاصل تقسیم دو عدد را باز می‌گرداند، در صورتی که عدد دوم برابر با صفر به جای تقسیم، پیام خطای مناسبی به کاربر نمایش دهد که امکان تقسیم بر صفر نیست:

function divide(a, b){
    if(b == 0){
        alert("امکان تقسیم بر صفر وجود ندارد");

        // خروج از اجرا
        return;
    }

    var result = a / b;
    return result;
}

شکل دیگر تعریف تابع

می‌توان توابع را به شکل زیر تعریف کرد:

var sum = function(a, b){
    return a + b;
};
