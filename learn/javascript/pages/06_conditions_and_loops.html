دستورات شرطی و تکرار

هدف اصلی JS اضافه کردن قابلیت‌های پویا می‌باشد. جهت اجرای تعداد زیادی از دستورات و تکرارها، از مفهوم شرط ها و حلقه‌های تکرار استفاده می‌کنیم. دستور if-else برای نوشتن بسیاری از شرط ها استفاده می‌شود. ساختار دستور if-else به شکل زیر است.

if (condition) {
    //JavaScript statements to be executed  
} else {
    //JavaScript statements to be executed 
}

برای مثال به تکه کد زیر را مد نظر قرار دهید:

if (i > 5) {
    x = 50;
}
else {
    x = 100;
}

تفسیر تکه کد بالا بسیار ساده است. اگر مقدار متغیر i از 5 بزرگتر باشد، عدد 5 به متغیر x انتساب داده خواهد شد و اگر مقدار متغیر i کمتر از 5 باشد سپس مقدار متغیر x برابر با 100 خواهد شد.

گاهی اوقات، لازم داریم که تعداد مشخصی از یک دستور اجرا شود، مثلاً نیاز به کدی داریم که 10 بار اجرا شود. برای این مورد از حلقه‌های تکرار استفاده می‌کنیم. هر حلقه‌ی تکرار دارای یک دستور جهت تکرار و یک شرط تکرار یا تعداد دفعات تکرار است.

JS دارای دو حلقه تکرار متدوال به شرح زیر است:

  • for : یک حلقه شمارنده که به تعداد مشخص دستوری را اجرا می‌کند.
  • while: یک حلقه تکرار که دارای یک شرط است و مادامی که این شرط برقرار باشد؛ تکرار حلقه ادامه خواهد داشت.

دستور for

ساختار دستور for به شکل زیر است:

for(مقدار افزایشی/کاهشی متغیر شمارنده ; شرط تکرار ; مقدار دهی اولیه متغیر شمارنده){
    // دستور جهت تکرار
}
  • مقدار دهی اولیه: یک متغیر با مقدار اولیه
  • شرط تکرار: یک شرط
  • مقدار افزایشی: دستوری که مقدار متغیر شمارنده را افزایش/کاهش می‌دهد

برای مثال دستور زیر را مشاهده فرمایید:

for(var i=0 ; i < 10 ; i++){
    alert(i);
}

دستور بالا 10 بار دستور alert(i) را اجرا می‌کند.

همانطور که از کد بالا مشخص است، در ابتدای حلقه تکرار یک متغیر شمارنده به نام i با مقدار اولیه 0 تعریف شده است. در قسمت شرط حلقه، شرط i < 10 در هر بار تکرار چک می‌شود که اگر مقدار متغیر i از 10 کوچکتر بود دستور افزایشی اجرا شود. در قسمت افزایشی، دستور i++ یک واحد به آن اضافه می‌کند.

جهت درک بهتر به جدول زیر دقت فرمایید:

مرحله تکرار شمارنده شرط تکرار نتیجه شرط کد اجرا شده
تکرار اول i=0 i < 10 true alert(0)
تکرار دوم i=1 i < 10 true alert(1)
تکرار سوم i=2 i < 10 true alert(2)
تکرار چهارم i=3 i < 10 true alert(3)
تکرار پنجم i=4 i < 10 true alert(4)
تکرار ششم i=5 i < 10 true alert(5)
تکرار هفتم i=6 i < 10 true alert(6)
تکرار هشتم i=7 i < 10 true alert(7)
تکرار نهم i=8 i < 10 true alert(8)
تکرار دهم i=9 i < 10 true alert(9)
تکرار یازدهم i=10 i < 10 false از حلقه خارج میشود!

💡 حلقه‌های تکرار for می‌تواند به شکل معکوس نوشته شود.

for(var i=10; i >= 0 ; i--){
    // دستورات
}

حلقه بالا از مقدار 10 تا مقدار 0 شروع به شمردن مکوکس می‌کند.

دستور while

ساختار دستور while به شکل زیر است:

while(شرط){
    // دستورات
}

دستور while مادامی که شرطش برقرار باشد تکرار می‌شود. مثلاً دستور زیر را در نظر بگیرید:

var a = 0;
while(a < 10){
    alert(a);
    a = a + 1;
}

تکه کد بالا 10 بار اجرا می‌شود. دستور while هر نوع شرطی را قبول می‌کند، برای مثال اگر نیاز به حلقه‌ای داشته باشیم که بی‌نهایت تکرار شود کافیست تکه کد زیر را بنویسیم:

while(true){
    // some code to repeat
}

کلمات کلیدی break و continue

این دو دستور؛ تنها در حلقه‌های تکرار قابل استفاده هستند و هر دوی آنها رفتار حلقه‌های تکرار را تغییر می‌دهند.

  1. break: سبب خروج از حلقه می‌شود
  2. continue: حلقه را به ابتدای آن منتقل می‌کند

تکه کدهای زیر را در نظر بگیرید:

var a = 0;
while(true){

    if( a == 1000)
        break;

    a++;
}

حلقه به علت وجود شرط true بصورت بی‌نهایت تکرار می‌شود. داخل حلقه شرطی نوشته شده است که اگر a == 0 شد، دستور break اجرا شود. پس حلقه‌ی بالا 1000 بار تکرار می‌شود و وقتی شرط a == 1000 برقرار شد، به علت وجود دستور break از حلقه خارج می‌شود.

for(var i=0;i<10000;i++){
    if ( i % 2 == 0) continue;

    // do some code
}

حلقه‌ی تکرار بالا از 0 تا 10000 اجرا می‌شود. داخل حلقه شرطی نوشته شده است که اگر باقیمانده مقدار شمارنده i بر عدد 2 صفر شد (یعنی اگر عدد زوج بود) ادامه دستورات حلقه اجرا نشود و مجدد به ابتدای حلقه برود. وجود دستور continue سبب این رفتار می‌شود.
