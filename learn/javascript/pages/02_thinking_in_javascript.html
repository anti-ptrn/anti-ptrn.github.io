تفکر در JavaScript

JavaScript یک زبان اسکریپت نویسی برای وب می‌باشد. قبل از اینکه وارد ریزه‌کاری‌های JavaScript شویم باید بدانیم که چرا از JS استفاده می‌شود. همانطور که HTML به مرورگر شما می‌گوید که صفحه باید چه شکلی باشد، JS برای تولید محتوای پویا[1] و اضافه کردن کارکردها به صفحه استفاده می‌شود.

برای شروع به کار و آشنایی با JS موارد زیر مورد نیاز است:

  • دانش ابتدایی از HTML
  • یک ویرایشگر متن مانند Notepad
  • یک مرورگر وب که JS روی آن فعال باشد. ( پیشنهاد می‌شود از Google Chrome استفاده کنید )

JS برای اجرای کد نیازی به کامپایلر [2] ندارد. کدهایی که به زبان JS نوشته شده‌اند را می‌توان داخل یک صفحه HTML قرار داد تا مرورگر وب کد JS را تفسیر و در زمان بارگذاری صفحه آن‌را اجرا می‌کند.

در این کتاب برای آموزش JS، از روش عملی استفاده می‌کنیم، که سبب افزایش بینش ما نسبت به JS می‌شود. پس از اتمام کتاب، با مفاهیم JS آشنا شده‌ایم و می‌توانیم از JS جهت پیاده‌سازی در برنامه‌های واقعی استفاده کنیم.

پاورقی

  1. صفحات وب به دو حالت ایستا - static و پویا - dynamic تقسیم می‌شوند. صفحات static صفحاتی هستند که هیچ تعاملی با کاربر ندارند و ثابت و بدون تغییر می‌مانند، اما صفحات dynamic با کاربر تعامل دارند و محتوای آن طبق برنامه نوشته شده تغییر می‌کند. ↩︎

  2. Compiler یا کامپایلر نرم‌افزاری است که کدهای نوشته شده به یک زبان برنامه‌نویسی را به زبان قابل اجرا برای کامپیوتر تبدیل میکند. مطالعه بیشتر در ویکیپدیا ↩︎

  3. برنامه‌نویسی شی‌گرا یا Object-oriented programming یا به اختصار OOP ↩︎
