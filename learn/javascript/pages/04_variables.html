متغیرها

در JS ، برای اهداف پردازشی نیازی به محلی برای ذخیره سازی مقادیر و اطلاعات داریم. به این محل‌های نگهداری اطلاعات در حافظه RAM متغیر یا Variable می‌گویند. می‌توان یک مقدار عددی [1] یا یک رشته [2] را در یک متغیر قرار داد. رشته‌های می‌بایست بین دو علامت " قرار بگیرند. همچنین مفهوم دیگری به نام ثابت یا constants وجود دارد. وقتی به یک ثابت مقداری انتساب داده می‌شود این مقدار دیگر قابل تغییر نیست و بصورت دائمی است این کاملاً برخلاف متغیرهاست که مقادیر آن‌ها قابل تغییر است.

خب ببینیم چطور می‌توان متغیر و ثابت در JS تعریف کرد:

var amount;
var amount=5;
const a;

بخوبی مشخص است که برای تعریف متغیر از کلمه کلیدی var و برای ثابت‌ها از کلمه کلیدی const استفاده می‌شود. همچنین در کد بالا می‌بینیم، می‌توان در زمان تعریف متغیر مقداری به آن منتسب کرد.

💡 const با تمامی مرورگرهای مدرن به غیر از Internet Explorer سازگار است.

یک متغیر می‌تواند بصورت محلی [3] و یا بصورت [4] سراسری تعریف شود. بیایید نگاهی بیندازیم که چطورمی‌توان فهمید که یک متغیر local/محلی و یا global/سراسری است.

⚠️ بخطار داشته باشید JS حساس به حروف کوچک و بزرگ است [5] است. بنابراین متغیر با نام myAttribute با myattribute یکسان نیست. نکته بسیار مهم دیگر این است که هر دستور در JS با یک ; به اتمام می‌رسد.

متغیر محلی و سراسری

متغیرهای سراسری در تمامی نقاط script قابل دسترسی هستند؛ این درحالیست که متغیرهای محلی فقط داخل تابع [6] ای که تعریف شدند قابل دسترسی هستند.

برای درک بهتر مفهوم بالا بیایید به تکه کد زیر نگاهی بیندازیم:

<!DOCTYPE html>
<html>

<head>
    <title> Introduction to JavaScript </title>
</head>

<body>
    <h1> Variables in JavaScript </h1>
    <script type="text/javascript">

        function subtraction(a, b) {
            var c;
            c = a - b;
            return c;
        } 

        var z = subtraction(10, 7); 
        
        document.write("After subtraction, the difference is " + z);
    </script>
</body>

</html>

در کد بالا؛ متغیرهای a و b به عنوان آرگومان ورودی به تابع ارسال شده اند. a و b متغیرهایی با دسترسی محلی هستند که تنها داخل بدنه تابع یا بلاکی که در آن تعریف شده اند، قابل دسترسی هستند. اما متغیر z بیرون تابع، در بدنه تگ script تعریف شده است، پس دسترسی آن بصورت سراسری می‌باشد و در تمامی کد قابل دسترسی است.

💡 در JS هر بلاک کد قسمتی از کد است که بین علامت‌های { و } قرار می‌گیرد.

اگر کد بالا را اجرا کنیم؛ خروحی به شکل زیر خواهیم داشت:

توابع جلوتر در تحت عنوان توابع در ادامه کتاب شرح داده خواهند شد، فعلاً بحث مربوط به متغیرها را ادامه می‌دهیم.

برای درک بهتر محدوده دسترسی [7] و مفهوم متغیرها، بایید نگاهی به کد زیر بیندازیم:

<html>
<head>
    <title> Scope of variables </title>
</head>

<body>
    <script type="text/javascript">
        var a = "Welcome to JavaScript";
        var b = "Introduction";

        function ShowAlert() {
            var a = "JavaScript";
            alert(a); 
            alert(b);
        }

        ShowAlert();
        alert(a);
    </script>
</body>
</html>

در زمان اجرا کد، میبینم که سه alert با ترتیب زیر نمایش داده می‌شوند.

  1. Welcome to JavaScript
  2. Introduction
  3. JavaScript

از نتیجه بالا می‌توان به خوبی متوجه شد که توابع می‌توانند به متغیرهای عمومی مانند alert اول و متغیرهای سراسری مانند alert دوم دسترسی داشته باشند. همچنین متغیرهایی که داخل تابع یا هر بلاک کد دیگر تعریف می‌شوند، در بیرون از آن قابل دسترس نیستند مانند alert آخر.

⚠️ توجه کنید، در مثال بالا در بیرون از تابع یک متغیر سراسری با نام a تعریف شده و در داخل تابع نیز متغیرمحلی با نام a تعریف شده است. به این دلیل که متغیر دوم در سطح داخلی تابع تعریف شده و هم نام متغیر سراسری a است، در داخل function نمی‌توان به مقدار متغیر سراسری a دسترسی داشت. بصورت ساده‌تر متغیر داخلی a متغیر سراسری a که در بیرون از تابع تعریف شده را مخفی کرده است.

انواع متغیر

همانطور که بالاتر اشاره گردید متغیرها نوع داده‌ای خاصی را در خود نگهداری می‌کنید.

در JS انواع داده‌ای پشتیبانی شده به شرح زیر است:

نوع نام انگلیسی مقدار
بولین Boolean true یا false
نال-پوچ null null
عددی Number 4 - 3.14
رشته String 4 - این یک رشته است
شی Object در ادامه درباره نوع آجکتت و اشیاء صحبت خواهد شد.

همانطور که قبلاً اشاره گردید برای تعریف متغیر از کلمه کلیدی var استفاده می‌کنیم. شاید از خود بپرسید نوع متغیر چطور تعیین میشود؟ جواب این سوال را با یک مثال می‌دهیم.

// عدد
var age = 27.5;
// بولین
var isMale = true;
// رشته‌ای
var name = "Ali";
// مقدار null
var children = null;

همانطور که از مثال‌های بالا پیداست، وقتی مقداری به یک متغیر در JS منتسب می‌شود، نوع متغیر به نوع مقداری که به آن وارد شده است تغییر پیدا میکند. از این جمله می‌توان این برداشت را داشت که یک متغیر می‌تواند در ابتدا مقدار number داشته باشد و در ادامه کد مقدار string بخود بگیرد. به کد زیر توجه کنید:

var myVariable = 27.5;
myVariable = null;
myVariable = "some string";
myVariable = false;

alert(myVariable);

در صورت اجرای کد بالا خواهید دید بدون هیچ خطایی اجرا می‌شود و در نهایت پیام false را به کاربر نمایش می‌دهد.

💡 با استفاده از کلمه کلیدی typeof در JS می‌توانید نوع متغیر را بدست آورید.

var myVariable = "Hello";
alert( typeof myVariable);

دستور بالا، مقدار string در قالب یک پیام نمایش می‌دهد.
