مقدمه

در این کتاب از مثال‌های کاربردی برای آموزش استفاده شده است و این امر سبب جلوگیری از سردرگمی خواننده در انبوهی از تئوری‌ها می‌شود. برای استفاده از این کتاب به جای یادگیری کلمه به کلمه کدها و دستوارت می‌توانید کدها و مثال‌ها را تغییر دهید و خروجی تغییر را مشاهده نمایید؛ این کار به شما در درک ظرافت‌های زبان JavaScript کمک می‌کند.

پس از مطالعه کتاب، با تمرین و تلاش بیشتر می‌توانید دانش بیشتری کسب کنید و صحبت به زبان وب[1] را شروع کنید.

پاورقی

  1. در اینجا نویسنده از JS به عنوان زبان وب نام برده است. امروزه از این زبان هم برای تولید برنامه‌های سمت سرور و هم سمت کلاینت استفاده زیاده می‌شود. در این لینک در وبلاگ آقای اسکات هنسلمن مقاله جالبی با عنوان JavaScript is Assembly Language for the Web آماده است. ↩︎

  2. برنامه‌نویسی شی‌گرا یا Object-oriented programming یا به اختصار OOP ↩︎
