نحوه استفاده از جاوا اسکریپت

به جای اتلاف وقت و سرگردان شدن در تئوری، بلافاصله با یک کد ساده آغاز می‌کنیم.

برای فهم اینکه JS چطور در یک صفحه وب استفاده می‌شود، به تکه کد زیر دقت کنید:

<!DOCTYPE html>
<html>

<head>
    <title> آشنایی با JavaScript </title>
    <script type="text/javascript">
        alert(" به جاوااسکریپت خوش آمدید ");
    </script>
</head>

</html>

اگر نگاهی به کد بیندازیم، خواهیم دید کد شامل خط <script type="text/javascript"> در بخش <head> است. تگ[1] script و خصوصیت[2] type به مروگر می‌گوید که کد JavaScript در HTML صفحه اضافه کرده ایم. همچنین می‌توان دستور alert را نیز در کد مشاهده کرد. بعداً در رابطه با دستور alert در کتاب صحبت خواهیم کرد. با اجرای کد، خروجی زیر را در صفحه خواهیم دید:

همانطور که دیدیم، کد JS می‌بایست بین تگ‌های <script> و </script> قرار بگیرد.

کد JS می‌تواند هم در بخش <head> و هم در بخش <body> قرار بگیرد. مساله انتخاب است و یا بستگی به صفحه وب دارد. اگر تعداد زیادی فایل CSS یا فایل‌های تصویری در صفحه وجود دارند، ترجیح این است که کد JS را در انتهای تگ <body> قرار دهیم. با اینکار اطمینان حاصل می‌کنیم که صفحه و تمام منابع آن ( مانند متن، تصاویر، استایل‌ها [3] ) بصورت کامل قبل از اجرای کد JS بارگذاری شده اند. هر نوع منبع به غیر از JS در صفحه بصورت غیرهمزمان[4] بارگذاری میشوند؛ این در حالی است که در هر لحظه فقط یک فایل JS دریافت می‌شود و این برای کارایی و سرعت صفحه مناسب نیست.

باید بدانیم که اگر نیاز به نوشتن مقدار زیادی از کد است، کار اصولی این است که کد JS را از کد HTML جدا کنیم. در این کتاب، برای راحتی، کدها را بین تگ <script> در صفحه HTML قرار داده ایم؛ اما به خاطر داشته باشید در برنامه‌های واقعی، فایل جداگانه‌ای با پسوند .js برای کدهای JS ایجاد می‌کنند. در اینجا ما یک فایل JS با نام main.js ایجاد کرده ایم و به صورت زیر می‌توانیم فایل مذکور را در کد فراخوانی کنیم:

<!DOCTYPE html>
<html>

<head>
    <title> آشنایی با JavaScript </title>
    <script type="text/javascript" src="main.js"></script>
</head>

</html>

در فایل main.js تنها تک کد زیر را می‌نویسیم:

alert(" به جاوااسکریپت خوش آمدید ");

وقتی کد را اجرا کنیم، نتیجه یکسانی با آنچه که در مثال قبلی مشاهده کردیم خواهیم دید. همانطور که می‌بینم برای اجرای main.js در صفحه HTML از کد زیر استفاده کردیم:

<script type="text/javascript" src="main.js"></script>

در اینجا فایل main.js می‌بایست در مسیری که فایل HTML قرار گرفته است، قرار بگیرد. اگر فایل JS ما در پوشه‌ی دیگری بود در خصوصیت src تگ script مسیر نسبی فایل را در آن می‌نویسیم.

تا اینجا با روش‌های فراخوانی و یا استفاده کردن کد JS در صفحه HTML آشنا شدیم.

پاورقی

  1. ساختاری از کد HTML یا XML هستند که بین علامت‌های < و > قرار می‌گیرند. ↩︎

  2. خصوصیت/صفت یا attribute به صفاتی می‌گویند که برای یک تگ تعریف می‌شوند، مثلاً در کد <body dir='rtl'></body> تگ body شامل صفت dir با مقدار rtl است. ↩︎

  3. کدهای CSS که برای تغییرات ظاهر مثل رنگ؛ اندازه و … در یک صفحه وب استفاده می‌شوند. ↩︎

  4. دریافت غیرهمزمان یا asynchronously به حالتی می‌گویند که مثلاً چندین منبع مثل تصویر بصورت همزمان دریافت شوند نه به صورت ترتیبی. یعنی مثلاً اگر 5 تصویر وجود دارد؛ آن 5 تصویر باهم دریافت شوند نه اینکه اول صبر کن یک تصویر کامل شود سپس به سراغ تصویر بعدی برود. ↩︎

  5. برنامه‌نویسی شی‌گرا یا Object-oriented programming یا به اختصار OOP ↩︎
