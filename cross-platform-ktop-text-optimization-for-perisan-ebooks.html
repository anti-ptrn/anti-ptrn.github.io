Anti-Pattern 💾 محلی برای اشتراک دانش برنامه‌نویسی، مهندسی نرم افزار، ابزارها اشتراک دانش برنامه‌نویسی

نسخه 1.5.1 برنامه KTOP (بهینه‌ساز کتاب‌های فارسی و عربی برای کیندل) قابل اجرا روی ویندوز، لینوکس و مک

علی بهرامی‌نژاد - 19-06-1395 - زمان تقریبی مطالعه: 2 دقیقه
Amazon Kindle eReader epub Calibre KTOP

ویرایش:

درصورتی که نسخه 1.5b را پیش‌تر دریافت کرده‌اید، لطفاً نسبت به دانلود نسخه 1.5.1 که لینک دانلود آن در همین پست آمده است اقدام کنید.

اگر 2 پست قبلی درباره معرفی برنامه KTOP و نحوه اضافه کردن فونت سفارشی به Calibre را مطالعه کرده باشید؛ با برنامه KTOP آشنایی دارید. KTOP (تلفظ کنید کِتاپ) بهینه‌ساز کتاب‌های فارسی و عربی برای کتابخوان آمازون Kindle می‌باشد.

بعد از انتشار این نرم‌افزار چندین درخواست مبنی بر ارائه نرم افزار برای سیستم‌عامل‌های Mac یا Linux دریافت کردم. امروز بالاخره موفق شدم و بهتر بگم فرصتی پیدا کردم که برنامه را با DOTNet Core سازگار کنم. با اینکار امکان اجرا بر روی سیستم‌عامل‌های لینوکس، مک و ویندوز فراهم شد.

تغییرات نسخه 1.5:

  1. اضافه شدن پشتیبانی از Mac و Linux
  2. ساخت Installer برای سیستم عامل Windows
  3. ساده‌تر شدن سوییچ ها
  4. حذف کتابخانه ویراستیار که جهت اصلاح نویسه‌ها استفاده میشد.

arabic-- بهینه‌سازی کتاب

سیستم عامل ویندوز

بعد از نصب، بصورت خودکار توسط برنامه آدرس KTOP در PATH ویندوز تعریف میشود و امکان استفاده فراگیر از آن در تمام محیط سیستم‌عامل به راحتی فراهم میشود.

جهت اجرا کافیست Command Prompt ویندوز رو باز کنید و دستور زیر رو بنویسد:

KTOP.CLI.exe "مسیر فایل شما" [--persian]

KTOP.CLI.exe "d:/my-books/programming/csharp.epub"

سیستم عامل مک و لینوکس

./KTOP.CLI "./csharp.epub"

نیازمندی ها

نسخه جدید برنامه به شکل self-contained منتشر شده است که جهت اجرا در سیستم‌عامل‌های مختلف نیاز به هیچ نیازمندی قبلی نیست، البته ممکنه در برخی از سیستم‌عامل‌ها ( مثل Ubuntu 14.04 ) برنامه در زمان اجرا با خطا مواجه شود جهت رفع مشکل DOTNet Core متناسب با سیستم‌عامل خود را نصب کنید.
